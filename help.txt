HoPi (Hospital Interface) Usage : node hopi <command>

[COMMAND]

- node hopi [login] [username] [password]
- node hopi [logout]
- node hopi [show] [pegawai] # menampilkan semua daftar pegawai
- node hopi [show] [pasien] # menampilkan semua daftar pasien
- node hopi [add] [pegawai] [nama] [posisi] [username] [password]
- node hopi [add] [pasien] [nama] "diagnose" # (value diagnose dipisahkan dengan tanda ',')
\_ contoh : node hopi add pasien corona "demam,batuk,panas 38C,bokek"
